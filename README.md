# 抽签器应用 (Lottery Picker App)

一个精美的抽签应用，可以帮助用户从列表中随机抽取选项。适用于抽奖、随机选择、决策等多种场景。

![抽签器应用截图](https://placeholder.svg?height=400&width=600)

## 特点

- 🎯 随机抽签功能，支持自定义抽签时间
- ✨ 精美的动画效果和抽签结果庆祝特效
- 📋 支持手动添加单个选项
- 🔢 支持根据数字区间批量生成选项
- 🎨 响应式设计，在各种设备上都有良好体验

## 安装与运行

### 前提条件

- Node.js 18.x 或更高版本
- npm 或 yarn 包管理器

### 安装步骤

1. 克隆仓库到本地
2. 下载依赖
```bash
npm install
```

3. 运行应用
```bash
npm run dev
```
